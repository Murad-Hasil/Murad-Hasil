## What I Do

I build AI systems that automate repetitive business workflows — like customer support, internal operations, and data processing.

The goal is simple: replace manual work with systems that run automatically, so teams can save time and reduce operational effort.

---

## How I Work

Before writing code, I break the system down clearly — defining how it should work, mapping different scenarios, and structuring the build step-by-step.

This avoids broken solutions, reduces rework, and ensures the system works reliably in real-world use.
