## What I Do

I build AI systems that automate repetitive business workflows — customer support, internal operations, and data processing.

Before writing code, I break the system down clearly — defining how it should work, mapping different scenarios, and structuring the build step-by-step using spec-driven development. This means every project starts with a written specification, so requirements are clear before a single line of code is written.

The result: fewer surprises, less rework, and systems that work reliably in production.
